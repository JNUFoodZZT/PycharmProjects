Food Detection

The project consists of two folders scripts and data.
In the data folder there must be the training and test datasets. For every sample type there must be a '_train.csv', and '_test.csv' in the ./data/ folder.

This experiment was implemented in Python 2.7.

To run the experiment you have to install the following required python libraries:

scikit-learn
matplotlib
pandas
pandas_ml
statsmodels

To run the experiment run the experiment.py file.
